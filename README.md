# Pinball-Emporium

## Pinball Emporium is a Frontend software for Windows PCs.
## It supports the following Pinball Software
+ Pinball FX 2023 (Steam)
+ Pinball FX3 (Steam)
+ Pinball FX2
+ Future Pinball
+ Future Pinball PinEvent
+ Visual Pinball 10.7
+ Visual Pinball Pinup Pup Packs
+ Zaccaria Pinball (Steam)
